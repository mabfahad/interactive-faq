== Description ==

Interactive FAQ Manager is a WordPress plugin that allows you to create and manage FAQs with categories, order numbers, and status control. It helps organize FAQs interactively on your website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/interactive-faq-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the FAQ menu to add, edit, and manage FAQs.

== Frequently Asked Questions ==

= How do I add a new FAQ? =

Go to the FAQ menu in the WordPress admin panel and click "Add New". Fill in the question, answer, select categories, order number, and status.

= Can I reorder FAQs? =

Yes, use the Order Number field to set the display order of the FAQs.
